Media Queries

Estamos aplicando media queries. Las media queries son una característica de CSS (Cascading Style Sheets) que permite aplicar estilos específicos en función de las características del dispositivo que está visualizando el sitio web. Esto se utiliza principalmente para hacer que el diseño de un sitio web sea responsive, es decir, que se adapte a distintos tamaños de pantalla, resoluciones y orientaciones de dispositivos.
